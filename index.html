Smart Aquarium Management System (SAMS)

AI-Powered Fish Disease Detection & Aquarium Management Platform

2024 – 2025

An AI-powered mobile application developed as my final-year capstone research project. It enables real-time fish disease detection using computer vision, supports post-mortem diagnosis, and provides intelligent treatment recommendations to improve aquarium health management.

YOLOv11 PyTorch React Native Flask Node.js Express.js Python
  • Real-time fish disease detection using a custom-trained YOLOv11 model.
  • AI-powered post-mortem diagnosis based on visible symptoms.
  • Treatment recommendations, quarantine alerts, and water quality suggestions.
  • Cross-platform mobile application built with React Native.
  • REST APIs developed with Flask and Node.js.
